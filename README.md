# Club Deportivo Árabe Unido — App (Demo)

Demo de la app del Club Deportivo Árabe Unido (Colón, Panamá · LPF 2026).

## Secciones
- **Partidos** — próximos encuentros de la Liga Panameña de Fútbol
- **Boletas** — compra de entradas por zona (General, Preferencia, VIP, Palco)
- **La Polla** — predicción de 4 datos por partido para ganar puntos
- **Mis puntos** — acumulación y canje por boletas o merch oficial

## Stack
HTML5 + CSS3 + Vanilla JavaScript. Sin frameworks ni build tools.
Diseño inspirado en la estética de [tiptap.dev](https://tiptap.dev).

## Correr localmente
```bash
python3 -m http.server 8137
# abrir http://localhost:8137
```
